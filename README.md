# GitHub Patterns Marketplace 🚀

O ecossistema definitivo para repositórios GitHub de alta performance, agnóstico de linguagem e focado em longevidade.

## 🕹️ Como usar (Quick Start)

Abra o **GitHub Copilot Chat** em um novo projeto e cole:

> "Aja como o Agente **Repo Architect**. Use as instruções de: `https://raw.githubusercontent.com/weslleycapelari/github-patterns/main/agents/setup-orchestrator.agent.md` e o registro em `https://raw.githubusercontent.com/weslleycapelari/github-patterns/main/registry.json`. Inicie o onboarding."

## 🏗️ O que este Marketplace oferece

- **Phase A-E Setup**: Automação completa do guia GitHub 100%.
- **6 Metodologias de Gestão**: Do Solo ao Enterprise (via `@issue-ops`).
- **Engenharia de Elite**: Injeção automática de regras de TDD, Imutabilidade e KISS.
- **Governança via CLI**: Prompts prontos para blindagem de branch (Phase B).
